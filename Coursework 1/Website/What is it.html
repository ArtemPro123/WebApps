What Does it stand for?

FTP is an acronym for "File Transfer Protocol". FTP is used to transfer files between computers on a network.

SSH, stands for "Secure Socket Shell", its a network protocol that provides administrators with a secure way to access a remote computer.

Details

File Transfer Protocol (FTP) is a standard network protocol used to transfer computer files. FTP uses separate control and data connections between the client and the server.

Secure Socket Shell provides strong authentication and secure encrypted data communications between computers connected over the Internet. SSH is widely used by administrators for managing systems and applications remotely, allowing them to connect to another computer over a network.

How it Works

FTP

Files can be transferred between two computers using FTP software. The user's computer is called the local host machine and it should connected to the Internet. The second machine, called the remote host, is also running FTP software and is also connected to the Internet.

FTP works in the same way as HTTP for transferring Web pages from a server to your computer.

Step 1: The local host machine connects to the remote host's IP address.

Step 2: The user would then enter a username/password.

Step 3: FTP software may have a GUI, which will allow the users to drag and drop files between the remote and local host machines.

If not, a series of FTP commands are used to log in to the remote host and transfer files between the machines.

Common Uses

FTP

FTP is most commonly used to download a file from a server using the Internet or to upload a file to a server.

SSH

Apart from Microsoft Windows, SSH software is included by default on most operating systems and it is used to connect remotely to another computer on the network.
